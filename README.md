# -Hash-Generator
Una herramienta web sencilla y potente para generar hashes en múltiples algoritmos: MD5, SHA-1, SHA-256, SHA-512, y más. Ideal para desarrolladores y entusiastas de la seguridad. Interfaz limpia, responsive y fácil de usar. Convierte texto o archivos a hash en tiempo real. Incluye opción de comparar hashes y soporte para salteado básico.
