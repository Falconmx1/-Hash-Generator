> Una herramienta web poderosa para generar hashes en múltiples algoritmos con soporte para archivos, salt y comparación de hashes.

## ✨ Características

- 🔒 **Múltiples algoritmos**: MD5, SHA-1, SHA-256, SHA-512
- 📝 **Hash de texto**: Genera hashes desde cualquier texto
- 📁 **Hash de archivos**: Procesa archivos TXT, JSON, CSV, código fuente
- 🧂 **Soporte para Salt**: Añade una capa extra de seguridad
- ⚖️ **Comparador de hashes**: Verifica si dos hashes coinciden
- 🎨 **Interfaz moderna**: Diseño limpio, responsive y con pestañas
- ⚡ **Tiempo real**: Generación instantánea
- 📋 **Copiar al portapapeles**: Un click y listo

## 🚀 Instalación y uso

### Requisitos previos
- Python 3.8 o superior
- pip (gestor de paquetes de Python)

### Paso a paso

```bash
# 1. Clonar el repositorio
git clone https://github.com/Falconmx1/-Hash-Generator.git
cd -Hash-Generator

# 2. Instalar dependencias
pip install -r requirements.txt

# 3. Ejecutar la aplicación
python app.py

# 4. Abrir el navegador en
http://localhost:5000
